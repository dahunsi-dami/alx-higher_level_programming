7. Module that validates an integer.
=====================================

How to use ``BaseGeometry`` module
----------------------------------

>>> BaseGeometry = __import__("7-base_geometry").BaseGeometry

Create instance of BaseGeometry:
>>> bg = BaseGeometry()

>>> bg.integer_validator("name", "Dre")
